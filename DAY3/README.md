# 로또 번호 생성기

1~45 중 중복 없는 6개의 번호를 무작위로 생성하는 Python CLI 스크립트입니다.
사용자가 원하는 게임 수 (1~5)까지 입력이 가능합니다.
1~45 중에서 제외하고 싶은 번호를 설정할 수 있습니다.

## 요구 사항

- Python 3

## 사용법

```bash
# 1게임 생성
python3 lotto.py

# 5게임 생성
python3 lotto.py -n 5

# 특정 번호를 제외하고 생성 (쉼표로 구분)
python3 lotto.py -n 5 -e 1,7,23
```

## 옵션

| 옵션 | 축약형 | 설명 | 기본값 |
| --- | --- | --- | --- |
| `--count` | `-n` | 생성할 게임 수 | `1` |
| `--exclude` | `-e` | 생성에서 제외할 번호 (쉼표로 구분) | 없음 |

## 구조

```
lotto.py
├── generate_game(exclude)  # 제외 번호를 뺀 풀에서 6개를 뽑아 정렬해 반환
└── main()                  # argparse로 옵션을 파싱하고 게임 수만큼 결과 출력
```
