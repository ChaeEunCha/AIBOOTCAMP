# BMI 계산기

키와 몸무게를 입력하면 BMI(체질량지수)를 계산하고, 결과에 따라 체형을 5단계로 분류해 시각적으로 보여주는 Python GUI 프로그램입니다.

## 실행 방법

Python 3와 `tkinter`가 필요합니다.

```
python3 bmi_gui.py
```

> macOS 시스템 기본 `python3`는 오래된 Tk(8.5) 버전이 포함되어 있어 입력창이 제대로 보이지 않는 렌더링 버그가 있을 수 있습니다. 이 경우 Homebrew 등으로 설치한 최신 Python(Tk 8.6 이상)으로 실행하세요.
>
> ```
> brew install python-tk
> python3.11 bmi_gui.py
> ```

## 주요 기능

- **BMI 계산**: 몸무게(kg)와 키(cm)를 입력받아 BMI 값을 계산
- **체형 분류**: 저체중 / 정상 / 과체중 / 비만 / 고도비만 5단계로 분류
- **시각화**: 분류 결과에 따라 몸 모양(머리+몸통)의 색상과 너비가 달라지는 캔버스 그림 표시
- **입력 검증**: 숫자가 아니거나 0 이하의 값을 입력하면 오류 메시지 표시

## BMI 분류 기준

| BMI 범위 | 분류 |
| --- | --- |
| 18.5 미만 | 저체중 |
| 18.5 ~ 22.9 | 정상 |
| 23.0 ~ 24.9 | 과체중 |
| 25.0 ~ 29.9 | 비만 |
| 30.0 이상 | 고도비만 |

## 기술 스택

- Python 3
- `tkinter` (GUI, 표준 라이브러리)
