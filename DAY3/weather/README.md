# 오늘 외출 컨디션 알리미

날씨 API(OpenWeatherMap)와 미세먼지 공공데이터(에어코리아)를 합쳐서, 오늘 외출하기 좋은 날인지 알려주는 Python GUI 프로그램입니다.

## 실행 방법

Python 3와 `tkinter`가 필요합니다.

```
python3 weather_gui.py
```

> macOS 시스템 기본 `python3`는 오래된 Tk(8.5) 버전이 포함되어 있어 입력창이 제대로 보이지 않는 렌더링 버그가 있을 수 있습니다. 이 경우 Homebrew 등으로 설치한 최신 Python(Tk 8.6 이상)으로 실행하세요.
>
> ```
> brew install python-tk
> python3.11 weather_gui.py
> ```

## API 키 설정

`.env` 파일에 아래 두 키를 넣어야 합니다.

```
weather_api="OpenWeatherMap API 키"
airkorea_api="에어코리아 일반 인증키(Decoding)"
```

- **weather_api**: [OpenWeatherMap](https://openweathermap.org/api)에서 무료 발급
- **airkorea_api**: [공공데이터포털](https://www.data.go.kr)에서 "한국환경공단_에어코리아_대기오염정보" 검색 → 활용신청 후 발급되는 **일반 인증키(Decoding)** 값 사용
  - 승인까지 몇 분~1일 소요될 수 있습니다.
  - 인증키에 `+`, `/`, `=` 같은 특수문자가 포함될 수 있으니 앞뒤 공백/따옴표만 제거하고 그대로 붙여넣으면 됩니다(URL 인코딩은 코드에서 처리).

## 주요 기능

- **날씨 조회**: 도시명을 입력하면 현재 기온, 체감온도, 날씨 상태를 표시
- **미세먼지 조회**: 시/도를 선택하면 해당 지역 측정소의 PM10·PM2.5 실시간 농도와 등급(좋음/보통/나쁨/매우나쁨)을 표시
- **종합 판정**: 날씨(강수·폭염·한파)와 미세먼지 등급을 종합해 **외출 추천 / 외출 주의 / 외출 비추천** 3단계로 판정하고, 판정 근거를 함께 표시

## 판정 기준

| 조건 | 판정 |
| --- | --- |
| 비/눈/뇌우, 기온 0℃ 이하 또는 33℃ 이상, 미세먼지 매우나쁨 | 외출 비추천 |
| 기온 5℃ 이하·30℃ 이상, 미세먼지 나쁨 | 외출 주의 |
| 위 조건에 모두 해당하지 않음 | 외출 추천 |

## 기술 스택

- Python 3
- `tkinter` (GUI, 표준 라이브러리)
- OpenWeatherMap API (날씨)
- 공공데이터포털 에어코리아 대기오염정보 API (미세먼지)
