# 취업 준비 To-Do List

취업 준비 과정(지원, 자기소개서, 코딩테스트, 면접, 스터디 등)의 할 일을 관리하기 위한 단일 HTML 파일 웹 앱입니다.

## 실행 방법

별도의 설치나 서버 없이 `index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

```
open index.html   # macOS
```

또는 파일 탐색기에서 `index.html`을 더블클릭하세요.

## 주요 기능

- **할 일 추가**: 제목, 카테고리, 마감일을 입력해 할 일을 등록
- **카테고리 분류**: 📝 지원 / ✍️ 자소서 / 💻 코딩테스트 / 🎤 면접 / 📚 스터디 / 📌 기타
- **마감일 관리**: 마감일이 지난 할 일은 기한초과로 빨간색 표시
- **필터**: 전체 / 진행중 / 완료 / 카테고리별로 목록 필터링
- **진행 현황 통계**: 전체 할 일 수, 완료 수, 기한초과 수, 전체 진행률(%) 표시
- **완료 체크 / 삭제**: 체크박스로 완료 처리, × 버튼으로 삭제
- **자동 저장**: 브라우저 `localStorage`에 저장되어 새로고침·재접속 후에도 데이터 유지

## 기술 스택

- HTML / CSS / JavaScript (Vanilla)
- 별도 프레임워크나 백엔드 없이 브라우저에서 단독 실행

## 데이터 보관 위치

모든 데이터는 서버로 전송되지 않고, 접속한 브라우저의 `localStorage`에만 저장됩니다. 브라우저 데이터를 삭제하거나 다른 브라우저/기기로 접속하면 데이터가 공유되지 않습니다.
