# PRD: cowork

## 개요
- **제품명**: cowork
- **대상 사용자**: 작은 팀 (3~10인)
- **핵심 가치**: 한 페이지에서 같이 보고 같이 쓰기

## 문제 정의
작은 팀은 문서 공유와 실시간 협업에 무거운 툴이 필요하지 않다. cowork는 팀원들이 하나의 페이지를 동시에 보고, 동시에 편집할 수 있는 가장 단순한 협업 공간을 제공한다.

## 필수 기능 (MVP)
| 기능 | 설명 |
|---|---|
| 페이지 생성 | 새로운 협업 페이지를 만든다 |
| 제목 수정 | 페이지 제목을 수정한다 |
| 본문 수정 | 페이지 본문을 수정한다 |
| 삭제 | 페이지를 삭제한다 |
| 실시간 동기화 | 여러 사용자가 동시에 같은 페이지를 보고 편집할 때 변경 사항이 즉시 반영된다 |
| 이메일 로그인 및 회원가입 | 이메일 기반으로 계정을 생성하고 로그인한다 |

## 제외 기능 (Out of Scope)
- 권한 관리 (역할/접근 권한 구분)
- 이미지 업로드
- 댓글

이 기능들은 팀 규모(3~10인)를 고려했을 때 초기 버전의 복잡도를 높이는 반면 핵심 가치 검증에는 필수적이지 않아 제외한다.

## 성공 기준
- 팀원 여러 명이 동시에 한 페이지를 열었을 때, 한 명의 수정 사항이 다른 사람 화면에 지연 없이 반영된다.
- 신규 사용자가 이메일만으로 가입하고 첫 페이지를 만드는 데까지 걸리는 시간이 짧다.
