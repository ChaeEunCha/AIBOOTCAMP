# PRD: cokanban

## 1. 개요

**cokanban**은 3~10인 규모의 작은 팀이 한 페이지에서 함께 업무 진행 현황을 확인하고 관리할 수 있는 칸반보드 서비스다.

## 2. 대상 사용자

- 소규모 팀(3~10명): 스타트업, 사이드 프로젝트 팀, 소규모 부서 등
- 별도 온보딩 없이 바로 사용 가능한 가벼운 협업 도구를 원하는 사용자

## 3. 핵심 가치

한 페이지 안에서 팀 전체가 실시간으로 업무 현황을 함께 보고 관리할 수 있다.

## 4. 필수 기능 (Must-have)

### 4.1 보드 구조
- 프로젝트 단위로 보드를 생성
- 보드 내 기본 목록: **할 일 / 진행 중 / 완료**

### 4.2 카드
- 카드 항목: 제목, 내용, 담당자, 우선순위
- 드래그 앤 드롭으로 목록 간 카드 이동 가능

### 4.3 실시간 동기화
- 한 사용자의 변경 사항(카드 생성/이동/수정)이 같은 보드를 보는 팀원 화면에 즉시 반영

### 4.4 인증
- 이메일 기반 로그인/회원가입

## 5. 제외 기능 (Out of scope)

- 권한 관리(역할/멤버별 접근 제어)
- 파일 업로드
- 댓글 기능
